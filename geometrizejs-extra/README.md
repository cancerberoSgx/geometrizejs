## Features

 * JPG, PNG, TIFF, GIF, BMP input formats
 * SVG, JSON, JPEG, PNG output formats
 * optimized SVG ouptut
 * ImageRunner steps iteration implementation
 * API to control the steps iteration  

## Install

